# Immich Kiosk Webhook Demo

## Requirements
- Python
- [UV](https://docs.astral.sh/uv/)

## Installation
1. create a virtual environment
  ```sh
  uv venv
  ```
2. activate the virtual environment
  ```sh
  source .venv/bin/activate
  ```
3. install the required packages
  ```sh
  uv sync
  ```
4. run the demo
  ```sh
  python main.py
  ```
